# Examples

Annotated session transcripts showing Claude Code patterns in action. Each example is a realistic conversation with annotations explaining what's happening and why each decision matters.

## How to Use These

1. **Read before your first session** — internalize the rhythm before you need it
2. **Reference when stuck** — if a session is going sideways, find the relevant example
3. **Compare your sessions** — if your results differ, check which patterns you're skipping
4. **Contribute your own** — real sessions with real problems are the most valuable examples
