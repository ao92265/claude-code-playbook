# Resources

Drop-in assets to use inside your projects: example session transcripts, CLAUDE.md templates for various stacks, and hook scripts for CI enforcement.

## What's in here

- **[Examples]({{ site.baseurl }}/examples/)** — three annotated real sessions (bug fix, new feature, multi-agent orchestration)
- **[Templates]({{ site.baseurl }}/templates/CLAUDE/)** — 11 CLAUDE.md templates for TypeScript, React, Node, Python, Go, Rust, Mobile, DevOps, Java, C#, and full-stack
- **[Hooks]({{ site.baseurl }}/hooks/)** — 9 shell scripts for pre-commit and pre-push safety (TypeScript, ESLint, Prettier, OOM-safe builds, secret detection, path protection)
