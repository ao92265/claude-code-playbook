# Claude Mythos Preview Is Here (I Reviewed The System Card)

**Source:** Joe Njenga, Claude Mythos Preview system-card review (forwarded to Alex 2026-04-08). Short-URL: `medium.com/p/351bf4445c9d`.

## Key takeaways

- **Claude Mythos** previewed on **Google Vertex AI** but **restricted** due to capability concerns.
- **A full tier above Opus** in raw capability.
- System-card analysis suggests **cybersecurity** capabilities drove the restriction.
- Njenga describes it as **"a pivotal moment in AI/cybersecurity"**.
- Relevant for model-routing strategy — Mythos isn't available today but will reshape the top of the tier when it does ship broadly.

## What Mythos is

From the system-card walkthrough: a Claude model **a full tier above Opus** — capability gains that were significant enough that Anthropic restricted broader release while running additional alignment and safety testing.

Exposed as a preview on Google Vertex AI but **not generally available** — access gated. Stays on limited release while Anthropic tests cyber safeguards.

## The cybersecurity angle

The system-card analysis suggests cybersecurity capabilities are what drove the restriction. Combined with the **Opus 4.7 shipping the first cybersecurity safeguards** (see [Opus 4.7 Reference]({{ site.baseurl }}/docs/opus-4-7/)), the pattern is visible: Anthropic is deliberately slowing down cyber-relevant capability releases until safeguards ship.

## Why this matters for model-routing strategy

If you're making a multi-year decision on model routing:

- **Opus 4.7** is the top of the generally-available tier today
- **Claude Mythos** is the next tier up but not available
- When Mythos ships broadly, the routing calculus changes — but you don't need to architect for it today
- Continue the 4.7 migration now; plan to re-evaluate when Mythos GA lands
