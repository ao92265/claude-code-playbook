# Claude Code /powerup: 10 Built-In Lessons That Teach You Features You Are Already Missing

**Source:** Reza Rezvani, "Claude Code /powerup: 10 Built-In Lessons..." (2026-04-03). Short-URL: `medium.com/p/7f5d222080b7`.

## Key takeaways

- **Interactive in-terminal tutorial** — not a docs link. Animated lessons rendered inside Claude Code's React + Ink UI.
- Requires **Claude Code v2.1.90+**.
- Ten built-in lessons revealing features most daily users miss — concise, interactive, ships inside the product.
- Rezvani (author) documented all 10 lessons in detail; worth a 30-minute evening read-through for any daily Claude Code user.

## What `/powerup` actually is

A new meta-command introduced in v2.1.90 that launches an interactive, terminal-native tutorial. Animated demonstrations run *inside the terminal* using Claude Code's React + Ink framework. Each of the 10 lessons covers one feature, with working examples you run as you learn.

> "Not a docs link. A real terminal-native course."

## Who it's for

- Existing Claude Code users who want to make sure they're using everything available
- Engineers who didn't read the changelog for the last six versions
- Team leads who want a consistent baseline of feature knowledge across their team

Run `/powerup` in any Claude Code session on v2.1.90+ and work through all 10 lessons.

## Why it matters for teams

`/powerup` is effectively a built-in onboarding path that doesn't rely on docs or tribal knowledge. Pair with `/team-onboarding` (see [that article]({{ site.baseurl }}/docs/news/team-onboarding/)) for a complete team-member ramp-up.
