## Project Basics

This is a Markdown documentation project (Claude Code Playbook for Force Information Systems). There is no TypeScript, no build step, and no tests. Changes are primarily `.md` files.

## Content Guidelines

- Target audience: developers new to Claude Code, often in enterprise/government settings
- Keep guides practical and actionable — copy-paste ready where possible
- Cross-link related docs (getting-started, onboarding, guide sections)
- When adding new docs, update the README.md directory tree

## Private Content

Do not commit `response-to-*.md` files — these are private correspondence drafts, not playbook content. They are gitignored.
